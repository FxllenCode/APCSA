# Opening

## Example

```java
public class CSA11 {
    public static void main(String[] args) {
        System.out.println("First expression: " + ( 3 + 2 / 4));
        System.out.println("Second expression: " + ( 23 % 5 + 8));
        System.out.println("Third expression: " + (  1.4 * 3 ));
        System.out.println("Fourth expression: " + ( 9 + 2 + " fish " + (3 - 1)));
        System.out.println("Fifth expression: " + ( 7 / (2 + 1) - 16));
    }
}
```

## Output

```
First expression: 3
Second expression: 11
Third expression: 4.2
Fourth expression: 11 fish 2
Fifth expression: -14
```

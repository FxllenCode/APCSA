# Math Expressions

## Opening

### Example 

```java
public class App {
    public static void main(String[] args) {
        int num1 = 15;
        int num2 = 12;
        num1 = num2;
        System.out.println("num 1 = " + num1);
        System.out.println("num 2 = " + num2);
    }
}
```

### Output

```
num 1 = 12
num 2 = 12
```
